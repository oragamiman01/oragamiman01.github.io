CraniUS

Hardware Engineering Intern

Designed PCB for CraniUS, a tech startup developing the world's first fully implantable device for direct delivery of medicine to the brain. My PCB design simplifies device programming by allowing for the power source to switch between regulated 3.3V power and battery power from the device. It features accessible pins and LED for debugging and ammeter support, plus a button to reset the connected device.

Speech enhancement

Class project (2022)

This project was for an audio signal processing class. To enhance speech, we used spectral subtraction, one of the first noise suppression algorithms. It consists of estimating the average noise signal and removing noise by subtracting the average noise spectrum from the main signal’s spectrum, resulting in a restored version of the original audio. One of the side effects of this noise suppression algorithm is the generation of musical noise which is formed because the noise spectrum generally changes during speech.

Computer vision guitar transcription

Class project (2023)

In this project, my group and I utilized different techniques to transcript guitar fingering positions from a video of a guitar player. My group considered two main approaches to this problem. The first of which is a deep learning CNN-based method using labeled frames as training data. The second method is a more traditional computer vision approach, breaking up the process into smaller steps. My main contributions to the project were the frame auto labeling pipeline and fretboard segmentation areas.

DTMF Dial Pad Encoding and decoding

Class project (2021)

This project was for my Signals and Systems class. It gave me an introduction into the elegance and power of signal processing. The goal of this project was to code a dual-tone multi-frequency dial pad encoder and decoder in Matlab. Using a combination of filtering and simple Fourier domain analysis, the digits from a 10-digit phone number can be decoded from a noisy audio signal. I enjoyed this class because it taught me how to think about signals in an applicable way.

Medical imaging estimation

Class project (2021)

In this project for a medical image analysis class, I designed an algorithm for left ventricle segmentation and stroke volume estimation from ultrasound sequence. The point of this project was to use a sequence of ultrasound images of the heart from two orthogonal views to identify the left ventricle and estimate the volume of blood pumped over one cycle of the heart. My partner and I at the time were not extremely familiar with deep learning methods, so we attempted to solve this problem through traditional image processing methods we learned in the class. This method proved to be less capable than deep learning segmentation techniques, but the process of designing an algorithm from scratch was a valuable learning experience to me.
